# Raycast QR Code Reader Extension

This Raycast extension allows you to read QR codes using your device's camera.

## Features

- Uses live camera feed to detect QR codes.
- Displays the decoded text from the QR code.
- Provides a button to copy the decoded text to the clipboard.

## Setup

1. Clone this repository or download the source code.
2. Navigate to the project directory in your terminal.
3. Run `npm install` to install the dependencies.
4. Run `npm run dev` to start the development server.

## Usage

1. Open Raycast.
2. Search for "Read QR Code from Camera".
3. Allow camera access if prompted.
4. Point your camera at a QR code.
5. The decoded text will be displayed.
6. Click the "Copy Text" button to copy the text to your clipboard.
